Key Specifications
🎥
Video Quality
Supports 1080p – 4K at 60–240 FPS. Minimal compression via FFmpeg. AVFoundation (iOS) + CameraX (Android).
⏱
Timing Precision
Frame-accurate to ±4ms. Uses frame index + FPS for millisecond timing. No GPS or manual stopwatch needed.
🔍
Pinch to Zoom
Pinch gesture zooms the video. Split clock overlays move with the zoom so timing stays pinned to the frame.
📱
Cross-Platform
Built with React Native or Flutter. One codebase for iOS + Android. Optional cloud sync for team storage.
